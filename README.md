# NeuroVue
NeuroVue is an AI-powered clinical diagnostics web platform that allows users to upload MRI brain scans and receive structured, AI-generated diagnostic insights powered by the Gemini API. Built with Next.js, React, and Tailwind CSS, the interface features a smooth step-by-step scrolling experience animated with Framer Motion — designed to feel closer to real clinical software than a typical AI tool. Diagnostic results can be exported instantly as a formatted, printable PDF report via jsPDF, eliminating manual copy-pasting.
